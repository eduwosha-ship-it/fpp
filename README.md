# FPP

## Installation Instructions
1. Clone the repository:
   
   ```bash
   git clone https://github.com/eduwosha-ship-it/fpp.git
   ```

2. Navigate to the project directory:
   
   ```bash
   cd fpp
   ```

3. Install the dependencies:
   
   ```bash
   npm install
   ```

4. Start the application:
   
   ```bash
   npm start
   ```
